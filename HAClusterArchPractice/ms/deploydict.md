#11.1.2.5	目录规划
|               |                            | 
|---------------|----------------------------| 
| 目录            | 含义                         | 
| /redis/bin    | redis可执行文件                 | 
| /redis/conf   | redis 和supervisord的配置文件    | 
| /redis/run    | redis和supervisord运行时的pid文件 | 
| /redis/log    | redis和supervisord的日志       | 
| /redis/script | 一些管理脚本和测试脚本                | 
| /redis/data   | Redis持久化数据目录               |
